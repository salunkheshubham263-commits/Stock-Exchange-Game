# Stock Exchange Game

The Stock Exchange Game is a web-based application that simulates real-time stock market behavior.  
The project fetches live stock prices using API calls and displays stock trends using charts.  
This project was developed as a group project to understand frontend–backend integration, APIs, and database usage.

---

## Features
- Fetches real-time stock prices using external APIs  
- Displays stock price trends using interactive charts  
- Stores data using a database  
- Simple and user-friendly UI  
- Web-based stock market simulation game  

---

## Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript  
- **Chart.js** (for stock price graphs)

### Backend
- Python  
- Flask  

### Database
- SQLite  

### Other Tools
- REST APIs  
- Git & GitHub

## How to Run the Project Locally
1. Install Python (for linux "sudo dnf install python3"
2. Install pip (for linux "sudo dnf install python3-pip")
1. Clone the repository:
   ```bash
   git clone https://github.com/salunkheshubham263-commits/Stock-Exchange-Game.git
2. cd Stock-Exchange-Game
3. pip install -r requirements.txt
4. python app.py
5. Open the browser and visit http://localhost:5000
